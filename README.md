# Lumina — Agence Web Premium ✦

Lumina est une agence digitale premium spécialisée dans la création de sites web haute performance qui convertissent les visiteurs en clients. Ce projet est une vitrine technologique alliant esthétique minimaliste (Bento design, Glassmorphism) et ingénierie de pointe.

## ✦ Points Forts

- **Design Ultra-Premium** : Interface moderne, épurée et responsive.
- **Performance de Pointe** : Optimisé avec Vite et React 19 pour des temps de chargement instantanés.
- **Animations Fluides** : Utilisation de `framer-motion` pour des transitions élégantes.
- **Internationalisation (i18n)** : Support complet du Français et de l'Anglais.
- **SEO Ready** : Structure sémantique et balises meta optimisées pour le référencement naturel.

## 🛠️ Stack Technique

- **Core** : [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Build Tool** : [Vite](https://vitejs.dev/)
- **Styling** : [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations** : [Framer Motion](https://www.framer.com/motion/)
- **Internationalisation** : [react-i18next](https://react.i18next.com/)
- **Icons** : [Lucide React](https://lucide.dev/)

## 🚀 Installation Locale

1. Clonez le dépôt
2. Installez les dépendances :

   ```bash
   npm install
   ```

3. Lancez le serveur de développement :

   ```bash
   npm run dev
   ```

## 🌐 Déploiement

Le site est automatiquement déployé sur **GitHub Pages** via GitHub Actions à chaque mise à jour sur la branche `main`.
